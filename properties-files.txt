# Configfiles
Below you will find all .properties files and their most relevant default values.

For some of them variables will be created in the playbook. These variables will be
used to template .j2 files from the templates folder.


## Server, consumer, producer, log etc
* /etc/kafka/server.properties
broker.id=0
num.partitions=1
zookeeper.connect=localhost:2181

* /etc/kafka/consumer.properties
bootstrap.servers=localhost:9092
group.id=test-consumer-group

* /etc/kafka/producer.properties
bootstrap.servers=localhost:9092
compression.type=none

* /etc/kafka-rest/kafka-rest.properties
id=kafka-rest-test-server
schema.registry.url=http://localhost:8081
zookeeper.connect=localhost:2181
bootstrap.servers=PLAINTEXT://localhost:9092


For the following files no variables will be changed.
* /etc/kafka-rest/log4j.properties
* /etc/kafka/zookeeper.properties
* /etc/kafka/log4j.properties
* /etc/kafka/tools-log4j.properties
